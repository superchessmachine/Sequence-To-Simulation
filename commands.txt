Some sample commands you can use:

Creating the GROMACS files:
gmx pdb2gmx -f input.pdb -o complex.gro -p topol.top -i posre.itp -ignh -ter

Defining the simulation box:
gmx editconf -f complex.gro -o boxed.gro -c -d 1.0 -bt dodecahedron

Solvating your system:
gmx solvate -cp boxed.gro -o solv.gro -p topol.top

Ionization configuration (ions_em.mdp):
integrator      = steep
emtol           = 1000
nsteps          = 500
cutoff-scheme   = Verlet
coulombtype     = PME
rcoulomb        = 1.2
vdwtype         = cut-off
rvdw            = 1.2
constraints     = h-bonds
pbc             = xyz

Preparing for Ionization:
gmx grompp -f ions_em.mdp -c solv.gro -p topol.top -o ions_NaCl.tpr -maxwarn 1

Ionization:
printf "SOL\n" | gmx genion -s ions_NaCl.tpr -o solv_ions.gro -p topol.top \
    -neutral -conc 0.15 -pname NA -nname CL

Energy Minimization Configuration (minim.mdp):
integrator      = steep
emtol           = 500
emstep          = 0.01
nsteps          = 50000
cutoff-scheme   = Verlet
coulombtype     = PME
rcoulomb        = 1.2
vdwtype         = cut-off
rvdw            = 1.2
constraints     = h-bonds
pbc             = xyz
DispCorr        = EnerPres

Preparing for Energy Minimization:
gmx grompp -f minim.mdp -c solv_ions.gro -p topol.top -o em.tpr

Running the Energy Minimization:
gmx mdrun  -deffnm em -ntmpi 1 -ntomp 16

NVT equilibration configuration (nvt.mdp):
integrator          = md
dt                  = 0.002
nsteps              = 50000       ; 100 ps
nstxout-compressed  = 5000
nstenergy           = 1000

cutoff-scheme       = Verlet
coulombtype         = PME
rcoulomb            = 1.2
vdwtype             = cut-off
vdw-modifier        = force-switch
rvdw                = 1.2
rvdw-switch         = 1.0

tcoupl              = V-rescale
tc-grps             = Protein Non-Protein
tau_t               = 1.0 1.0
ref_t               = 310 310

pcoupl              = no

constraints         = h-bonds
pbc                 = xyz
DispCorr            = EnerPres
define              = -DPOSRES

Preparing for NVT: 
gmx grompp -f nvt.mdp -c em.gro -r em.gro -p topol.top -o nvt.tpr

Running the NVT: 
gmx mdrun  -deffnm nvt -ntmpi 1 -ntomp 16

NPT equilibration configuration (npt.mdp):
integrator          = md
dt                  = 0.002
nsteps              = 500000      ; 1 ns
nstxout-compressed  = 5000
nstenergy           = 1000

cutoff-scheme       = Verlet
coulombtype         = PME
rcoulomb            = 1.2
vdwtype             = cut-off
vdw-modifier        = force-switch
rvdw                = 1.2
rvdw-switch         = 1.0

tcoupl              = V-rescale
tc-grps             = Protein Non-Protein
tau_t               = 1.0 1.0
ref_t               = 310 310

pcoupl              = C-rescale
tau_p               = 5.0
ref_p               = 1.0
compressibility     = 4.5e-5

constraints         = h-bonds
pbc                 = xyz
DispCorr            = EnerPres
define              = -DPOSRES
refcoord-scaling    = com

Preparing for NPT:
gmx grompp -f npt.mdp -c nvt.gro -r nvt.gro -t nvt.cpt -p topol.top -o npt.tpr

Running the NPT:
gmx mdrun  -deffnm npt -ntmpi 1 -ntomp 16
